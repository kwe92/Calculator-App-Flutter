# Neumorphic Calculator

A neumorphic calculator app: Dart Lang | Flutter SDK

## Concepts Covered:

- Showing widgets on screen
- Core OOP principles
- const Classes
- get => computed variables
- static methods and variables
- Modularity
- GridView.builder
- Writing functions
- Creating custom widgets
- if else in Dart
- setState()
- Widget state management
- Functions as first class objects
- Refactoring code overime

## Widgets Used:

Core Flutter Widgets:

- StatelessWidget
- StatefulWidget
- MaterialApp
- SafeArea
- Scaffold
- AppBar
- Text
- Column
- Container
- Expanded
- Gridview
- FloatingActionButton
- Center

Some Custom Widgets:

- MyApp
- CalculatorPage
- NeumorphicCalculatorScreen
- CalculatorButton(custom neumorphic buttons)
- EqualPressed

Dependencies:

- math_expressions ^2.0.0
- flutter_inset_box_shadow: ^1.0.8
